## Preparación


### En replit:

Crear un replit con node.js y ejecutar en el shell:


```
   $ npx create-react-app client
```

Luego ejecuta el cliente de este modo:

```
   $ cd client
   $ npm start
```


### En tu PC

```
  $ mkdir tutorial-client
  $ cd tutorial-client
  $ npm init
  $ npx create-react-app client
  $ cd client
  $ npm start
  ```

Detén el servidor presionando `ctrl-c`.
